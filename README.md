# Weather Diary

**Автор:** Артём Романов

**Описание:**
Простое GUI-приложение для ведения дневника погоды. Позволяет добавлять записи о погоде с датой, температурой, описанием и осадками. Есть фильтрация по дате и температуре. Данные сохраняются в JSON.

**Как запустить:**
1. Установите Python 3.x.
2. Склонируйте репозиторий.
3. Запустите: `python weather_diary.py`

**Пример использования:**
- Введите дату (например: 2024-06-01), температуру (например: 22.5), описание (например: "Солнечно") и выберите осадки.
- Нажмите "Добавить запись".
- Используйте фильтры для поиска записей по дате или температуре.
